# Bookstore
Simple REST API built with the MEAN Stack

#Technologies
Node 
Mongodb
Express
Angularjs

# How to run
Clone repo
 ``` shell
 git clone https://github.com/M0unir/Bookstore.git
 ```
Install dependencies
 ``` shell
 npm install && cd client && bower install
 ```
Run app
```shell
npm start
```
